| 명령어 | 설명 |
| --- | --- |
| `ls` | 현재 디렉토리에 있는 파일 목록을 출력합니다. |
| `cd` | 디렉토리를 변경합니다. |
| `mkdir` | 디렉토리를 생성합니다. |
| `touch` | 파일을 생성하거나 파일 수정 시간을 변경합니다. |
| `rm` | 파일 또는 디렉토리를 삭제합니다. |
| `cp` | 파일을 복사합니다. |
| `mv` | 파일을 이동하거나 이름을 변경합니다. |
| `cat` | 파일 내용을 출력합니다. |
| `more` | 파일 내용을 페이지 단위로 출력합니다. |
| `less` | 파일 내용을 스크롤 가능한 페이지 단위로 출력합니다. |
| `grep` | 특정 문자열을 검색합니다. |
| `find` | 파일을 검색합니다. |
